Проекты

Все мои проекты включают оптимизацию производства с использованием систем автоматизации управления технологическими процессами,создание чертежей для различных инженерных конструкций и систем,проведение анализа данных с целью выявления паттернов и оптимизации бизнес-процессов.

Сепаратор нефти

Программист

Участие в разработке сепаратора нефти для оптимизации процессов разделения нефтяных фракций в производственной среде. Тестирование и отладка программного обеспечения с целью обеспечения стабильной работы сепаратора в различных условиях эксплуатации.

Шкафы управления

Проектировщик

Участие в проекте по разработке чертежей и технической документации для шкафов управления, предназначенных для промышленного использования. Создание технических чертежей шкафов управления с использованием программного обеспечения AutoCAD

Обнаружение объектов и
распознавание

Программист

Разработка системы компьютерного зрения (Computer Vision) для обнаружения и распознавания объектов в реальном времени с использованием современных методов машинного обучения. Разработка и реализация алгоритмов компьютерного зрения с применением библиотек машинного обучения, таких как TensorFlow и подготовка и аугментация датасетов с использованием платформы Roboflow для обучения моделей машинного обучения.

Разработки ПО для
микроконтроллеров

Программист

Программного обеспечения для микроконтроллеров, предназначенного для управления и контроля различными устройствами и системами. Написание кода на языках программирования, таких как C, C++, с использованием интегрированных сред разработки (IDE) и SDK для конкретных микроконтроллеров.

Анализ данных

Программист

В проекте по анализу данных с целью извлечения ценной информации из имеющихся данных для принятия бизнес-решений или оптимизации процессов. Сбор, очистка и предобработка данных для подготовки к анализу. Визуализация результатов анализа с использованием инструментов визуализации данных (Matplotlib, Seaborn).
